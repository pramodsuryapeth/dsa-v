Pseudocode

                dp[i][w] = 0  for i = 0 or w = 0
                For each item i from 1 to n:
                    For each weight w from 1 to W:
                        If weight[i] <= w:
                            dp[i][w] = max(dp[i-1][w], dp[i-1][w - weight[i]] + value[i])
                        Else:
                            dp[i][w] = dp[i-1][w]
